## Short Recommendation

The safest current option is **Combination 2: OneAgent `1.329` + classic Lambda layer + OpenLLMetry/Traceloop `0.45.4`**.

Why:

- It showed prompt and completion content.
- It showed LangGraph workflow/task spans.
- It showed guardrail details in `traceloop.entity.output`.
- It did not show the latest-layer `4096` character UI truncation behavior in the copied classic sample.
- It matches the known internal baseline more closely than the latest-layer experiments.

Latest-layer native OneAgent instrumentation is promising, especially for inline guardrails, but it did not yet provide enough prompt/response/guardrail detail to replace the classic + Traceloop setup in this test.
